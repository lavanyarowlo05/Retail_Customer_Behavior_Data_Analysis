# 🛍️ Customer Behavior Data Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using SQL, Python, and Power BI. The goal is to extract meaningful insights from customer data to understand purchasing patterns, customer segments, and revenue trends.

---

## 🎯 Objective

* Analyze customer purchase behavior
* Identify key revenue drivers
* Segment customers based on buying patterns
* Evaluate the impact of discounts and subscriptions
* Build an interactive dashboard for business insights

---

## 🛠️ Tech Stack

* **SQL (PostgreSQL)** – Data querying and analysis
* **Python (Pandas, SQLAlchemy)** – Data processing & database integration
* **Power BI** – Dashboard creation and visualization
* **Jupyter Notebook** – Exploratory data analysis

---

## ⚙️ Process / Workflow

1. Data Collection (Customer dataset)
2. Data Cleaning & Preprocessing (Python)
3. Data Storage (PostgreSQL)
4. SQL Analysis (Business queries)
5. Visualization (Power BI Dashboard)
6. Insights & Reporting

---

## 🔍 Key Features

* Revenue analysis by gender and age group
* Customer segmentation (New, Returning, Loyal)
* Discount impact analysis
* Subscription behavior insights
* Top-performing products by rating and sales
* Shipping type comparison
* Category-wise product performance

---

## 📊 SQL Analysis Highlights

Some key business questions solved:

* Total revenue by gender
* Customers spending above average with discounts
* Top 5 highest-rated products
* Subscription vs non-subscription spending
* Discount usage percentage per product
* Customer segmentation using CASE statements
* Top products within each category using window functions
